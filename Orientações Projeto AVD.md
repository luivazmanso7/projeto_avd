y**Diretrizes do Projeto Final: Análise e Visualização de Dados (AVD)** 

**Disciplina:** Análise e Visualização de Dados com Python 

**1\. Visão Geral do Projeto** 

Este projeto desafia o aluno a percorrer o ciclo completo de um projeto de ciência de dados: desde a captura bruta de informações até a entrega de uma ferramenta interativa de tomada de decisão. O foco principal está na qualidade técnica da análise e na aplicação rigorosa de princípios de design e comunicação visual. 

**2\. Sequência Obrigatória de Desenvolvimento** 

**Fase 1: Extração de Dados (Web Scrapping)** 

Capturar dados de um ambiente web dinâmico ou estático. O aluno deve demonstrar proficiência em lidar com tags HTML e estruturas de dados não organizadas. 

**Fase 2: ETL & Data Wrangling** 

Processamento dos dados brutos utilizando a biblioteca Pandas. É obrigatório realizar: limpeza de valores nulos, tratamento de tipos, normalização e criação de métricas calculadas. 

**Fase 3: Análise Estatística Exploratória** 

Aplicação de funções estatísticas para descrever o fenômeno estudado. Inclui correlação de Pearson, análise de variância, quartis e identificação de outliers. 

**Fase 4: Visualização com Princípios de Gestalt** 

Construção de gráficos que não apenas exibam dados, mas contem uma história. O uso de cores, tamanhos e agrupamentos deve seguir as leis da percepção visual (Gestalt). 

**Fase 5: Dashboard Interativo (Streamlit)** 

Implementação do projeto em uma aplicação web funcional, permitindo que o usuário final interaja com as variáveis e filtros.
